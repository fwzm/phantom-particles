# 幻影粒子 | 交互式3D视觉系统

一个实时交互式的3D粒子可视化系统，支持手势识别和多种粒子形状变换。

## ✨ 功能特性

- 🎨 **实时3D粒子渲染** - 15000+ 粒子实时动画
- 👆 **手势交互** - 使用摄像头识别手势控制粒子
- 🔄 **形状变换** - 在不同粒子模式间流畅切换
- 🌊 **物理模拟** - 逼真的粒子扩散与吸引效果
- 📱 **响应式设计** - 适配各种屏幕尺寸

## 🚀 快速开始

### 线上体验
访问 [在线演示](https://your-github-username.github.io/phantom-particles)

### 本地运行
```bash
# 克隆仓库
git clone https://github.com/your-github-username/phantom-particles.git

# 进入目录
cd phantom-particles

# 用浏览器打开 index.html
```

## 🛠️ 技术栈

- **Three.js** - 3D渲染引擎
- **MediaPipe** - 手势识别
- **Vanilla JavaScript** - 无框架依赖

## 📖 使用说明

1. **允许摄像头权限** - 首次打开时浏览器会请求摄像头权限
2. **手势控制**
   - 👋 张开手掌：控制粒子缩放
   - ✌️ 两根手指：控制粒子扩散
3. **快捷键**
   - `H` - 心形模式
   - `T` - 字体模式
   - `Space` - 暂停/继续
